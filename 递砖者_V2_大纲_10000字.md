## 第一章：省心孩子（约12000字）

### 废墟图书馆Mod对应
- **敌人**：省心孩子×3 + 优等生×1
- **战场**：后巷筛选点
- **机制**：优绩主义的筛选——省心孩子不会主动攻击，只跟随优等生
- **核心隐喻**：被分配的角色不是真实的自我

### 时间线（2004-2010）

**2004年10月**
- 四川乡镇出生，农民家庭
- 不是"农村孩子"的浪漫化叙事——是真实的农民家庭：种地、打工、姐姐哭

**爷爷去世**
- 记忆模糊，不确定是真实悲伤还是植入的
- "我分不清哪些是真实感受，哪些是后来编造的"——认知裂缝的起点

**婆婆照顾到学龄前**
- 父母出去打工
- 姐姐在哭，我只是看着
- **核心场景**："姐姐哭，我在看。不是我没有感情，是我不知道应该有什么感情。"
- 这是"淡"的第一次显影——情感的早期关闭

**入县城小学**
- 被行政留级——不是成绩差，是年龄太小
- 这个细节很重要：不是因为笨被留级，是因为太小
- 但"被留级"的标记留在了身上

**"适应得快"**
- 老师、家长、亲戚都说"这孩子适应能力强"
- 不是适应，是第一次学会了"淡"
- 淡 = 不反应、不哭闹、不让大人操心 = 省心

**被分配的角色**
- 你是那个不需要被担心的
- 你是那个不哭的
- 你是那个省心的
- 你是那个成绩好的
- 用了整个童年把这个角色执行到完美

### 阶级底色（农民家庭的核心影响）

**侧写V4.0的分析**：
- 技术分层与权力真空："修bug不如换人快"——对"人是最贵也是最便宜的资源"的直觉认知
- 基本信任契约："先砌砖再谈钱"——农民工基本的生存逻辑
- 实用主义与反论："加钱加的最多"与"风评最高"的矛盾

**白桦的原话**：
- "我爸说：钱不是花出来的，是省出来的。我说：时间不是省出来的，是算出来的。我爸不说话了。"
- "我妈说：读书是唯一出路。我说：读书是出路之一，但出路不是唯一。我妈哭了。"

### 核心概念：省心孩子

**省心孩子是什么**：
- 一个被社会分配的角色
- 不是"孩子真的省心"，是"孩子知道不省心会被惩罚"
- 省心 = 不表达需求 + 不表达情绪 + 不让大人操心

**省心孩子的代价**：
- 情感的早期关闭（"淡"）
- 真实自我被角色覆盖
- 成年后无法区分"我是谁"和"我应该是什么样"

**省心孩子与递砖的关系**：
- 省心孩子只接砖，不递砖
- 递砖需要暴露自己（这是省心孩子不敢的）
- 但递砖真空要求：向不确定的接收者递出——这意味着暴露

### 本章结尾

"我用了整个童年学会省心。用了整个青春期学会不换内存。用了整个成年才学会递砖。

省心孩子是地基。地基上有裂缝。但墙还是从地基上砌起来的。

0。"

## 第二章：翼的学徒（约12000字）

### 废墟图书馆Mod对应
- **敌人**：翼的学徒×2 + 认知滤网测试体×2
- **战场**：滤网覆盖
- **机制**：认知干扰、感知屏蔽——从翼（系统）内部学习，同时被滤网过滤
- **核心隐喻**：独自发明、独自监测、独自命名

### 时间线（2010-2017）

**小学高年级**
- 发现学习能力周期性波动（约一周为单位）
- 有时候一周内掌握整章内容
- 有时候一周内连基本概念都记不住
- 不是"懒"或"不努力"，是真实的认知状态波动

**"换内存"系统的发明**

**2G状态**：
- 最低运行状态
- 只能维持基本功能（吃饭、睡觉、不惹事）
- 学习？做不了。社交？不想做。
- 但不是抑郁——是"内存不够，系统降频"

**4G状态**：
- 正常学习状态
- 可以听课、可以写作业、可以考试
- 成绩中等偏上
- 但不是"真实水平"

**8G状态**：
- 流体智力全开
- 认知密度极高
- 小学五年级直觉猜到勾股定理
- 中考前一个月从零开始学几何，全县前列
- "整个人都亮了"——但不可控

**"洗澡后会换内存"**
- 洗澡 = 物理上的"重启"
- 水温、水流、独处空间 = 状态切换的触发器
- 这个发现是白桦独自完成的
- 没有人可以讨论——"说出去谁信？"

**独自发明、独自监测、独自命名**
- 三个"我"的分工：
  - 一个负责维持成绩（优等生模式）
  - 一个负责逃避内疚感（省心孩子模式）
  - 一个负责接住罗持续单向递来的好感（不知所措模式）
- 不是分裂——是分工
- 看《杀了我治愈我》后意识到："原来有人跟我一样"

**罗的出现**

**罗是谁**：
- 整个青春期唯一主动靠近白桦的人
- 加QQ、撒娇、送耳机（九十元）
- 白桦查了价格——"可能没有她说的那么贵"
- 但礼物的价值不在价格，在"有人主动递砖"

**白桦的反应**：
- 打算回礼（二月生日）
- 但父亲失业——"可以稍微动一点"变成"绝对不能动"
- 选择：让她放手
- "我觉得自己配不上"
- "不是不爱，是觉得自己不配被爱"

**中考后的密室邀约**：
- 罗约白桦去密室
- 白桦："手机没电"
- 没有说"我不想去"，说"手机没电"
- 这是省心孩子的语言——不拒绝，只消失

**高中同班三年**：
- 不再主动说话
- 三年，同一个教室，零交流
- 罗递了砖，白桦拒收了

### 核心概念：翼的学徒

**翼是谁**：
- 不是具体的人
- 是"系统"——学校、家庭、社会、优绩主义
- 翼的学徒 = 从系统内部学习，同时被系统过滤

**滤网覆盖**：
- 系统让你看到它想让你看到的
- 你学会了系统的规则，但你的真实感受被滤掉了
- "淡" = 被滤网过滤后的情感输出

**换内存系统的意义**：
- 不是医学诊断——是民间智慧
- 一个12岁孩子独自发明的自我监测系统
- 从2G到4G到8G——不是病理，是生存策略

### 本章结尾

"我发明了换内存，不是为了控制状态，是为了理解自己。

理解为什么有时候我是8G的天才，有时候我是2G的废物。

理解为什么罗递了砖，我却说手机没电。

理解为什么我在翼的系统里学了那么多，却还是被滤网覆盖。

翼的学徒不是荣耀。是被系统利用而不自知。

但换内存是我自己的。谁也拿不走。

0。"

## 第三章：董事会（约12000字）

### 废墟图书馆Mod对应
- **敌人**：董事会代理×1 + 羽的护卫×2
- **战场**：制度性碾碎
- **机制**：政策清洗——被制度碾碎的不是能力，是坚持
- **核心隐喻**：标准化屠宰场——被迫完成无意义仪式的原型体验

### 时间线（2017-2020）

**高一入学**
- 分班选免费班——与罗同班
- 不再主动说话
- "我选了免费班，因为父亲失业了。不是因为我想和罗同班。"
- 但选了免费班 = 和罗同班 = 三年的沉默

**高一崩溃：失眠**
- 无法跟老师听课
- 无法看书自学
- 从小学一年级就在乡镇凭感觉拿第一的孩子
- 突然发现自己赖以为生的认知系统不再听使唤
- "不是内存从8G掉到2G。是整台机器都在说：跑不动了。"

**自救：网课**
- 第一次月考前找到B站数学教学视频
- 凌晨三四点还在听
- 拿到年级第一
- 不是逆袭——是自救
- "网课不需要身体好。只需要凌晨三四点还在听的人。"

**标准化屠宰场**
- 被迫学习无意义的东西
- 厌恶但必须完成
- A系统（规则）vs B系统（真相）
- A系统：你要完成所有仪式，拿到毕业证
- B系统：这些仪式没有意义，但你不能退出

**父亲失业**
- 只能选免费班
- 需要毕业证换入场券
- "我知道A系统是虚伪的，但我需要A系统的入场券"

**高二小测试：跌出年级前三十**
- 身体直接说：撑不住了
- 初二开始扛的所有东西：
  - 父亲失业
  - 耳机（罗的礼物，未回礼）
  - 免费班沉默
  - 内存波动（换内存系统超负荷）
  - 对日本的复杂情感（后来想深造的方向）
  - 对病的恐惧（还没确诊，但已经感觉到不对）
- "再也不能被一次网课、一次年级第一掩盖"

### 核心概念：董事会

**董事会是什么**：
- 不是学校管理层
- 是"制度"本身——抽象、无处不在、碾碎个体
- 董事会的决策"无关善恶，只关成本"

**制度性碾碎**：
- 碾碎的不是能力——是坚持
- 碾碎的不是成绩——是意义感
- 碾碎的不是身体——是"我为什么要继续"的答案

**诚实自限者的跌落**：
- 从年级第一到撑不住
- 不是堕落
- 是真相终于追上了

### 本章结尾

"董事会碾碎了我。不是因为我弱。是因为我撑了太久。

从初一到高二，五年。免费班、父亲失业、换内存、罗的耳机。

一次网课、一次年级第一，能盖多久？

高二那次跌出前三十，是真相终于追上了我。

不是堕落。是诚实。

0。"
